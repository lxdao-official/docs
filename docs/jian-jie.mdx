**LXDAO 致力于支持有价值的开源项目的正向循环。**

LX = 良心 = Conscience and Integrity，这体现了我们的核心价值观和文化，反映了我们的理想主义建设者群体的长期愿景，这也是开源项目支持者的共同特质。

LXDAO 秉持「开源共享、协作自治、可持续责任、学习实验、有良心」的**核心价值观，聚焦**研究、探索和支持有价值的 Web3 应用。

通过机制设计与协作实验，LXDAO 系统研究开源项目在资金、技术及治理等领域的核心问题；通过原型验证推进 Web3 前沿应用的可行性探索；同时，为早期项目提供 Grants、研发工具、品牌推广、生态合作关系等全方位的支持，助力优质开源项目的启动、成长和可持续运营。

我们聚焦于以下领域：

- **DAO 治理与建设**：我们推动去中心化自治组织的治理模式和技术建设。
- **Web3 教育**：我们致力于推广 Web3 技术和理念，培养更多的 Web3 建设者。
- **开源项目支持**：我们提供资金和人力支持，助力公共物品的可持续发展。

短期聚焦四大战略路径：

1）核心机制验证与高潜力项目孵化；

2）开源生态参与者网络扩展；

3）资金分配方式优化及可持续财务模型构建；

4）Web3 协作式学习平台的打造与建设，强化 LXDAO 在 Web3 开源领域的关键影响力。

长期愿景定位于成为 Web3 创新应用的「黄埔军校」及关键生态组件：通过搭建全球开源协作网络，建立覆盖项目孵化、融资支持、生态协作的全链路基础设施。我们希望连接全球开源力量，推动 Web3 生态的发展。
